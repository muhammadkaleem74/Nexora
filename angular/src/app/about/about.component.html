Platform information and documentation

This is a startup template built on ASP.NET Boilerplate and Module Zero. For an enterprise-grade starter, see ASP.NET ZERO.

What is ASP.NET Boilerplate?

ASP.NET Boilerplate is an application framework built on the latest ASP.NET Core. It simplifies authorization, dependency injection, validation, exception handling, localization, logging, caching, and background jobs — built on familiar tools like Entity Framework, AutoMapper, and Castle Windsor.

It implements N-Layer architecture (Domain, Application, Infrastructure, and Presentation) and Domain Driven Design (Entities, Repositories, Domain/Application Services, DTOs). It also provides infrastructure for common software development best practices.

What is Module Zero?

ASP.NET Boilerplate is designed to be independent of any database schema. It leaves some concepts abstract and optional — like audit logging, permission and setting stores — which require a data store.

Module Zero implements all fundamental concepts such as tenant management (multi-tenancy), role management, user management, authorization, setting management, language management, and audit logging.

Based on Microsoft ASP.NET Core Identity

Module Zero is based on Microsoft's ASP.NET Core Identity library. It extends user and role managers and implements user and role stores using generic repositories.
